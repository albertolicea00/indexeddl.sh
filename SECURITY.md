# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in **indexeddl.sh**, please report it responsibly:

1. **Do not** publicly disclose the issue.
2. Open a security advisory in the repository's GitHub Security tab: [GitHub Security Advisories](https://github.com/albertolicea00/indexeddl.sh/security) (replace with actual repository URL).
3. Include a clear description of the vulnerability, steps to reproduce, and any relevant details.

We will acknowledge receipt of your report within 2 business days and work with you to resolve the issue promptly.

## Policy Scope

This policy applies to all components of the `indexeddl.sh` project, including scripts, documentation, and any related services.

## Responsible Disclosure

We appreciate your help in keeping our software secure. Thank you for adhering to responsible disclosure practices.
